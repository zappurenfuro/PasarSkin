PASARSKIN - ASSETS FOLDER
===========================

Please add your images to this folder:

REQUIRED FILES:
---------------
1. logo.png - Your PasarSkin logo
   - Recommended size: 200x200px (transparent background)
   - Used in navbar and footer

2. hero-bg.jpg (OPTIONAL) - Hero section background image
   - Recommended size: 1920x1080px
   - Can be a CS2/gaming related image
   - Will be used as a subtle background overlay

TIPS:
-----
- Use PNG format for logos (supports transparency)
- Use JPG format for photos/backgrounds (smaller file size)
- Optimize images for web to improve loading speed
- Keep file names exactly as mentioned above

The website will work without these images, but will look much better with your actual logo and images!
